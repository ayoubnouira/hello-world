<h1 align=center>GIT COMMANDS</h1>
<ol>
  <li>Setup and configuration and help commands
    <ol>
      <li>git config --global user.name <strong>"Your Name"</strong></li>
      <li>git config --global user.email <strong>"you@example.com"</strong></li>
      <li>git config --global init.default branch main</li>
      <li>git config -h</li>
      <li>git help config
      <li>git help</li>
    </ol>
  </li>
  <li>clear the terminal
    <ol>
      <li>clear</li>
    </ol>
  </li>
  <li>change directory and create new git folder
    <ol>
      <li>cd <strong>c:/users/..rest of the file name</strong></li>
      <li>git init</li>
    </ol>
  </li>
  <li>basic git workflow
    <ol>
      <li>git status</li>
      <li>git add <strong>file name</strong></li>
      <li>git rm --cached <strong>file name</strong></li>
      <li>git add .;or;git add --all;or;git add -A</li>
      <li>git restore --staged <strong>file name</strong>
      <li>git commit -m <strong>"message about what you did"</strong></li>
      <li>git commit -a -m <strong>"message about what you did"</strong></li>
      <li>git diff</li>
      <li>git diff --staged</li>
    </ol>
  </li>
  <li>advantages of git
    <ol>
      <li>git restore <strong>"file name"</strong></li>
      <li>git mv <strong>"original file name"</strong> <strong>"new file name"</strong></li>
    </ol>
  </li>
</ol>
